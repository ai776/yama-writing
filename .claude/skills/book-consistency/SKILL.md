# book-consistency — 品質保証層 章間矛盾ゼロ化

原稿全体を通し読みし、4つの台帳を作成して章間の矛盾を機械的に検出する。
「前に言ってたことと違う」を完全に排除する。

## なぜ台帳が必要か

原稿を読むだけの矛盾チェックでは:
- AIのコンテキストウィンドウの制約で全章を同時に見られない
- 暗黙的な矛盾（用語の定義ズレ、数値の前提条件の違い）を見逃す
- 「どちらを採用すべきか」の判断基準がない

台帳を作成することで:
- 主張・定義・数値・ルールを構造化データとして比較できる
- 矛盾の検出が網羅的・機械的になる
- 統一案の作成に客観的な基準を使える

## Inputs / Outputs

- **In:** 完成原稿 `{書籍タイトル}.md`、`BookSpec.json`、`chapter_blueprints.md`、`heading_spec.md`
- **Out:** `consistency_report.md`（4台帳 + 矛盾候補ペア + 統一案）

## ワークフロー

1. **全章通し読み** — 原稿を最初から最後まで読み込む
2. **4台帳作成:**
   - **定義台帳** — 用語・概念の定義を章ごとに抽出
   - **ルール台帳** — 「〜の場合はこう」の条件分岐を抽出
   - **数値台帳** — 数字・前提条件・計算条件を抽出
   - **推奨台帳** — おすすめの順序・優先順位・推奨事項を抽出
3. **クロスチェック** — 各台帳の項目間で矛盾がないか照合
4. **矛盾候補の特定** — 矛盾の重要度を判定（CRITICAL / HIGH / MEDIUM / LOW）
5. **統一案の作成** — どちらを採用すべきかの基準と修正案を提示
6. **レビューチェーン:**
   - book-qa に提出（G13-Q: 台帳の網羅性、矛盾検出の正確性）
   - book-producer に提出（G13-P: 統一案の妥当性、戦略整合性）
   - 著者に提出（G13-H: 統一案の承認、採用判断）

## 4台帳の定義

### 定義台帳（Definition Ledger）
抽出対象:
- 専門用語の定義（「利回りとは〜」「出口戦略とは〜」）
- 概念の説明（「中古アパート投資の本質は〜」）
- 分類・カテゴリ（「物件には3種類ある」）

チェック観点:
- 同じ用語が章ごとに異なる意味で使われていないか
- 定義の範囲が章によって広がったり狭まったりしていないか

### ルール台帳（Rule Ledger）
抽出対象:
- 条件分岐（「〜の場合は〜すべき」）
- 判断基準（「〜が〜以上なら買い」）
- 手順・ステップ（「まず〜して、次に〜する」）

チェック観点:
- 同じ状況に対して異なるルールを提示していないか
- 例外条件の扱いが統一されているか

### 数値台帳（Number Ledger）
抽出対象:
- 具体的な数字（「利回り8%」「頭金300万円」）
- 前提条件（「地方都市の場合」「築20年の場合」）
- 計算式・計算条件（「表面利回り = 年間賃料 ÷ 物件価格」）

チェック観点:
- 同じ数字が異なる前提条件で使われていないか
- 計算の前提が章によって変わっていないか
- 概算値と正確な値が混在していないか

### 推奨台帳（Recommendation Ledger）
抽出対象:
- 推奨事項（「〜がおすすめ」「〜を推奨する」）
- 優先順位（「まず〜、次に〜」）
- 比較評価（「AよりBが良い」）

チェック観点:
- 推奨の優先順位が章によって変わっていないか
- 前半で推奨したものを後半で否定していないか
- 比較の基準が統一されているか

## 矛盾の重要度判定

| 重要度 | 基準 | 例 |
|--------|------|-----|
| CRITICAL | 核心メッセージに関わる矛盾 | 第1章「出口が最重要」vs 第5章「利回りが最重要」 |
| HIGH | 読者の行動判断に影響する矛盾 | 第3章「利回り8%以上」vs 第6章「利回り6%でもOK」 |
| MEDIUM | 用語定義・表現の不統一 | 第2章「キャッシュフロー」vs 第4章「手残り」（同義語の揺れ） |
| LOW | 細かい数値・表現の差異 | 第1章「約300万円」vs 第7章「300万円前後」 |

## 統一案の作成基準

1. **BookSpec の core_message に近い方を採用**
2. **より具体的・正確な記述を採用**（曖昧より具体的）
3. **後半の記述を優先**（著者の考えが洗練されている可能性）
4. **判断できない場合は著者に確認**（G13-H で判断を仰ぐ）
