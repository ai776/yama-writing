# 台帳作成ルール

## 1. 抽出の基本原則

### 原文忠実
- 著者の言葉をそのまま記録する（要約しない）
- 出典（章番号・節番号・行番号）を必ず記録する
- 文脈情報（条件・前提）も合わせて記録する

### 網羅性
- 全章を対象とする（はじめに・あとがきも含む）
- 暗黙の定義・ルールも抽出する（明示的に「〜とは」と書かれていなくても）
- 図表のキャプション・注釈も対象とする

### 一意性
- 同じ項目が複数箇所に出現する場合、すべての出現箇所を記録する
- 最初の出現を「主記述」、以降を「参照」とする

## 2. 台帳ごとの抽出パターン

### 定義台帳の抽出パターン

```
検出パターン:
- 「〜とは」「〜というのは」「〜を〜と呼ぶ」
- 「〜は〜のことだ」「〜は〜を意味する」
- 「〜には〜がある」（分類・列挙）
- 括弧内の説明（「キャッシュフロー（手残り）」）
- 初出時の注釈・説明文

記録項目:
| フィールド | 内容 |
|-----------|------|
| ID | D-001, D-002, ... |
| 用語 | 定義される用語・概念 |
| 定義内容 | 著者による定義（原文） |
| 出典 | 第X章 第Y節 L.ZZ |
| 文脈 | どのような話の流れで定義されたか |
| 関連用語 | 同義語・類義語（あれば） |
```

### ルール台帳の抽出パターン

```
検出パターン:
- 「〜の場合は〜する」「〜なら〜」
- 「〜すべき」「〜してはいけない」
- 「ポイントは〜」「コツは〜」
- 「Step 1: 〜、Step 2: 〜」
- 「〜が大切」「〜が重要」

記録項目:
| フィールド | 内容 |
|-----------|------|
| ID | R-001, R-002, ... |
| ルール内容 | 著者が示す判断基準・行動指針（原文） |
| 適用条件 | どのような状況で適用されるか |
| 例外条件 | 例外がある場合（あれば） |
| 出典 | 第X章 第Y節 L.ZZ |
```

### 数値台帳の抽出パターン

```
検出パターン:
- 数字を含む文（「8%」「300万円」「築20年」）
- 計算式（「年間賃料 ÷ 物件価格」）
- 比較数値（「AはBの2倍」）
- 範囲指定（「100万〜300万円」）

記録項目:
| フィールド | 内容 |
|-----------|------|
| ID | N-001, N-002, ... |
| 数値 | 具体的な数字 |
| 単位 | 円、%、年、件 等 |
| 前提条件 | その数値が成り立つ条件 |
| 計算根拠 | 計算式・出所（あれば） |
| 出典 | 第X章 第Y節 L.ZZ |
```

### 推奨台帳の抽出パターン

```
検出パターン:
- 「おすすめは〜」「〜を推奨する」
- 「まず〜すべき」「最初に〜」
- 「AよりBが良い」「AよりBを選ぶ」
- 「最も大切なのは〜」「一番重要なのは〜」
- 「〜を優先する」

記録項目:
| フィールド | 内容 |
|-----------|------|
| ID | P-001, P-002, ... |
| 推奨内容 | 著者の推奨（原文） |
| 優先順位 | 順位づけがある場合（1位、2位 等） |
| 対象読者 | 誰向けの推奨か（初心者/中級者 等） |
| 条件 | 推奨が成り立つ条件 |
| 出典 | 第X章 第Y節 L.ZZ |
```

## 3. クロスチェックの手順

### Step 1: 台帳内チェック（同一台帳内の矛盾）
- 同じIDカテゴリ内で、同一トピックに対する異なる記述を検出
- 例: D-003「利回り = 年間賃料 ÷ 物件価格」vs D-012「利回り = (年間賃料 - 経費) ÷ 物件価格」

### Step 2: 台帳間チェック（異なる台帳間の矛盾）
- 定義台帳 × ルール台帳: 定義と矛盾するルールがないか
- 数値台帳 × ルール台帳: 数値の前提とルールの条件が一致しているか
- 推奨台帳 × ルール台帳: 推奨がルールと矛盾していないか

### Step 3: BookSpec整合チェック
- 全台帳 × BookSpec.json: 核心メッセージ・戦略と矛盾する記述がないか

## 4. 矛盾ペアの記述フォーマット

```markdown
### 矛盾 #1 [HIGH]

**テーマ:** 目標利回り

**記述A:**
> 利回り8%以上の物件を探しましょう
— 第3章 第2節 L.145

**記述B:**
> 利回り6%台でも立地がよければ十分です
— 第6章 第3節 L.312

**矛盾の種類:** 数値の不一致（N-005 vs N-018）

**統一案:**
記述Bを採用。ただし以下のように修正:
「基本的には利回り8%以上を目安にしますが、立地条件が良い場合は6%台でも検討の余地があります」

**採用基準:** 後半の記述がより実践的で、初心者の行動判断に即している
```
