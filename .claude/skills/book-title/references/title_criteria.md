# タイトル評価基準・ガイドライン

## 4軸スコアリング

各候補を以下の4軸で10点満点評価する:

### 1. SEO適合性（検索されやすさ）

| 評価基準 | 点数 |
|---------|------|
| ペルソナの検索意図に合致するキーワードを含む | +3 |
| Amazon/書店での検索ヒットが見込める | +3 |
| 競合タイトルと差別化できている | +2 |
| 文字数が適切（メイン: 10〜25文字、サブ: 15〜35文字） | +2 |

### 2. 訴求力（手に取りたくなるか）

| 評価基準 | 点数 |
|---------|------|
| 具体的なベネフィットが伝わる | +3 |
| 好奇心・緊急性・共感のいずれかを喚起する | +3 |
| 記憶に残りやすい（リズム・語感） | +2 |
| 表紙デザインとの相性（視覚的映え） | +2 |

### 3. ペルソナ適合性（読者に刺さるか）

| 評価基準 | 点数 |
|---------|------|
| ペルソナの悩み・課題に直接言及している | +4 |
| ペルソナの語彙レベルに合っている | +3 |
| ペルソナが「自分向けだ」と感じる | +3 |

### 4. 市場差別化（埋もれないか）

| 評価基準 | 点数 |
|---------|------|
| 同ジャンルの既存書籍と被らない | +4 |
| 独自の切り口・フレーズがある | +3 |
| 著者のブランドと一致している | +3 |

## メインタイトルのパターン

| パターン | 例 | 効果 |
|---------|-----|------|
| ベネフィット直球 | 「〜で〜が変わる」 | わかりやすい |
| 問いかけ型 | 「なぜ〜なのか？」 | 好奇心喚起 |
| 数字入り | 「〜の7つの法則」 | 具体性 |
| 逆説型 | 「〜しなくていい」 | 意外性 |
| 物語型 | 「〜が教えてくれたこと」 | 共感 |

## サブタイトルの役割

- メインタイトルで伝えきれない情報を補足
- 対象読者やベネフィットの明確化
- SEOキーワードの補完
- 「誰に」「何を」「どうやって」のいずれかを補う

## 注意事項

- 情報源にない主張をタイトルに含めない
- 過度に煽る表現（「絶対」「確実に」「人生が変わる」）は避ける
- 著者の禁止表現に抵触しないこと（BookSpec参照）
