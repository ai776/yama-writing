# 制作条件ヒアリング項目

## 必須ヒアリング項目

以下の6項目が **すべて** 確認されるまで ProductionSpec 生成に進んではならない。

### 1. 情報源タイプ（source_type）

- セミナー文字起こし → `seminar`
- YouTube動画 → `youtube`
- 既存原稿のリボーン → `reborn`

#### YouTube版固有の追加ヒアリング

- 対象チャンネルURL
- 対象動画リスト（指定がある場合）
- 動画選定基準の優先順位

#### リボーン版固有の追加ヒアリング

- リボーンの目的・背景
- 維持したい核心メッセージ（魂）
- 新しいターゲット読者像（変更がある場合）

### 2. トーン＆スタイル

- 基本文体（です/ます調 or だ/である調）
- 口語の許容度（低/中/高）
- 著者の人柄を反映した語りのスタイル

### 3. 禁止表現

- 著者固有のNGワード
- 業界的に避けるべき表現
- 禁止トピック
- ※AI禁止フレーズは book-qa で別途担保

### 4. 分量設計

- 総文字数（例: 40,000〜45,000字）
- 章数（例: 8〜10章）
- 章ごとの文字数予算
- まえがき・あとがきの文字数

### 5. 図表・挿絵方針

- 挿入頻度（章ごと / 節ごと / 任意）
- 図のタイプ（フロー図 / 比較表 / イラスト指示 / Mermaid）
- 挿入位置ルール
- キャプション形式

### 6. 引用ルール

- source_type に応じて自動設定されるが、著者の希望で上書き可能

| source_type | デフォルト引用形式 |
|-------------|------------------|
| seminar | 発言者名、開始〜終了の行番号 or 章節名 |
| youtube | 発言者名、動画タイトル、タイムスタンプ `[mm:ss]` |
| reborn | 原稿の章/節番号、ページ番号 |
