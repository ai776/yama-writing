## 使い方

各コードブロックをMarkdownファイルにそのまま貼り付けるか、
Mermaid Live Editor (https://mermaid.live/) でプレビュー・画像化できる。

## テンプレート集

### フローチャート（プロセス説明用）

```mermaid
flowchart TD
    A[開始] --> B[処理1]
    B --> C[処理2]
    C --> D[終了]
```

### 比較表（Markdown表を使用）

```markdown
| 項目 | 従来の方法 | 新しい方法 |
|------|-----------|-----------|
| 特徴1 | {内容} | {内容} |
| 特徴2 | {内容} | {内容} |
```

### タイムライン

```mermaid
gantt
    title {タイムラインタイトル}
    dateFormat YYYY
    section フェーズ1
    イベント1 :2020, 2021
    section フェーズ2
    イベント2 :2021, 2023
```

### マインドマップ

```mermaid
mindmap
  root((中心テーマ))
    トピック1
      サブトピック1-1
      サブトピック1-2
    トピック2
      サブトピック2-1
    トピック3
```
