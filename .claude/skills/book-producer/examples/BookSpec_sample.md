## Part B: 制作条件

### 情報源タイプ

| 項目 | 内容 |
|------|------|
| source_type | seminar |

### トーン＆スタイル

- **基本文体:** です/ます調ベース、語りかけ風
- **口語許容度:** 中程度（「〜たんです」OK、極端な口語はNG）
- **禁止表現（著者固有）:** 「絶対に治る」「必ず改善する」（医療系の断定表現）

### 分量設計

| 項目 | 値 |
|------|-----|
| 総文字数目安 | 42,000字 |
| 章数 | 8章 |
| 章ごと文字数予算 | 4,500〜5,000字 |
| まえがき | 1,500字 |
| あとがき | 1,500字 |

### 図表・挿絵方針

| 項目 | 設定 |
|------|------|
| 挿入頻度 | 各章に2点程度 |
| 図のタイプ | 解剖図イラスト指示、フロー図（Mermaid） |
| 挿入位置ルール | 説明の直後に配置 |
| キャプション | 「図X-Y: タイトル」形式 |

### 引用ルール

**設定:** seminar形式 — 発言者名、開始〜終了の行番号 or 章節名
