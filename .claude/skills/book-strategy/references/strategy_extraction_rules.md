# 戦略ヒアリング項目

## 必須ヒアリング項目

以下の5項目が **すべて** 確認されるまで StrategySpec 生成に進んではならない。

### 1. 事業ゴール

- 書籍の最終目的は何か（LINE登録→商品販売 / 採用 / ブランディング / 認知拡大）
- バックエンド商品は何か
- CTA導線はどう設計するか（例: QRコード→LINE→ステップ配信→販売）

### 2. 想定読者（ペルソナ）

- 年齢層、性別、職業、立場
- 主な悩み・課題
- 価値観・行動パターン
- どんな場面で本を手にするか

### 3. KPI設計

- 何をもって書籍の成功とするか
- 測定指標（CV率 / LINE登録数 / 読了率 / Amazon評価 / 重版回数など）
- 流入経路（SNS / セミナー / 広告 / 口コミ）

### 4. 書籍の役割定義

書籍が事業のどこに位置づけられるかを明確にする:

| 役割 | 説明 |
|------|------|
| リード獲得装置 | LINE登録やメルマガ登録を促す |
| 権威性証明 | 著者の専門性を示す |
| 営業資料代替 | 商談時に渡す名刺代わり |
| ブランド構築 | 長期的な信頼を積み上げる |
| 教育コンテンツ | 顧客育成の一環 |

### 5. 核心メッセージ（魂）

- この本で一つだけ伝えるなら何か
- 読者に最も持ち帰ってほしいもの
- 一文で言い切れるか（言い切れないなら絞り込みが足りない）
