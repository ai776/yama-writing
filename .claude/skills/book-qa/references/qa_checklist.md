# QAチェック手順 詳細

book-qa Skill がチェック時に従う手順と判定基準の詳細。

---

## チェック手順

### Step 1: 参照ファイル読み込み（必須）

以下を **必ず** 読み込んでからチェックを開始する:
1. `style_manual.md` — AI生成文章の特徴と回避策の全文
2. `banned_phrases.md` — 禁止フレーズ抽出リスト
3. BookSpec.json の `tone.banned_expressions` — 著者固有の禁止表現

### Step 2: AI表現チェック

#### 2-1. 禁止フレーズ検出
- `banned_phrases.md` のリストに該当する表現を全件検出
- 「即修正」「要注意」「文脈判断」の3段階で判定

#### 2-2. 構造パターン検出
- **定型三段構成:** 「導入→展開→まとめ」が機械的に繰り返されていないか
- **語調の均一性:** 感情・主張の起伏が少なく、一定のリズムで続いていないか
- **構造的整いすぎ:** 不自然なほど整然としていないか

#### 2-3. 接続詞の過多
- 「まず」「次に」「そのため」「つまり」「要するに」等の出現頻度をカウント
- **判定基準:** 1,000文字あたり3回以上 → 「過多」

#### 2-4. 「あなた」の過度な使用
- 出現頻度をカウント
- **判定基準:** 1,000文字あたり2回以上 → 「過度」
- 連続する段落で繰り返されていないか

#### 2-5. 曖昧語尾
- 「〜かもしれません」「〜の可能性があります」「〜と考えられています」の連続使用
- 主張すべき箇所で曖昧語尾になっていないか

#### 2-6. 抽象語の連発
- 「価値創出」「持続可能」「課題解決」「多様性」等
- 具体例やエピソードなしで抽象論が続いていないか

#### 2-7. 類義語・冗長表現
- 「目標を達成するためのゴール」「重要で大切な役割」等の重複表現

### Step 3: 文体チェック

- 全体のトーンが BookSpec の設定と一致しているか
- 著者固有の禁止表現が含まれていないか
- 語尾のバリエーションは十分か
- 人間らしい揺らぎ（短文/主観/葛藤/比喩の新鮮さ）があるか

### Step 4: 校正・校閲

- 誤字脱字
- 文法的な誤り
- 表記の不統一（数字表記、敬称、専門用語）
- 情報源にない記述の混入（ファクトチェック）

### Step 5: 章間整合チェック（⑨ book-draft 対象時のみ）

旧 book-cohesion の機能を統合（現在は book-consistency が詳細チェックを担当）。全章を通し読みし以下を検証する:

#### 5-1. 論理的一貫性・矛盾
- 前半で述べた主張と後半で矛盾する記述はないか
- 用語の定義が章によって変わっていないか
- 因果関係の論理に飛躍はないか
- 同一の固有名詞・数字が章によって異なっていないか

#### 5-2. 重複
- 同じエピソード・事例が複数箇所で繰り返されていないか
- 同じ主張が異なる言い回しで重複していないか
- 意図的な繰り返し（核心メッセージの強調）との区別

#### 5-3. 構成・流れの自然さ
- 章から章への遷移はスムーズか
- 各章の末尾から次章の冒頭への接続は自然か
- 「ブリッジ文」（章間の繋ぎ）が適切に入っているか

#### 5-4. 見出しと本文の整合
- すべての章タイトル・小見出しが本文内容を正確に反映しているか
- 見出しから期待される内容と実際の本文にギャップはないか

#### 5-5. トーン・文体の統一性
- 全章を通して文体は一貫しているか
- 特定の章だけトーンが大きく異なっていないか

#### 5-6. 核心メッセージの一貫性
- BookSpec の core_message が全体を通して一貫して伝わるか
- 核心メッセージが薄まっている章はないか

**重要度判定:**
| 重要度 | 基準 |
|--------|------|
| HIGH | 読者が混乱する矛盾、核心メッセージの逸脱 |
| MEDIUM | 重複、トーンの揺れ、接続の不自然さ |
| LOW | 表記の不統一、軽微な見出し不整合 |

### Step 6: 人間らしさスコア（⑨ book-draft 対象時のみ）

以下の7項目を10点満点で自己評価:

| 評価項目 | 基準 |
|----------|------|
| 禁止フレーズの不在 | 検出0件=10, 1〜2件=7, 3件以上=4 |
| 構造の自然さ | 定型パターン0件=10 |
| 接続詞の適度さ | 過多判定0章=10 |
| 語尾のバリエーション | 3種以上の語尾=10 |
| 感情・個性の表現 | 主観・葛藤・ユーモアの存在 |
| 具体性 | 事例・数字・エピソードの充実度 |
| 読者との対話感 | 問いかけ・共感表現の自然さ |
