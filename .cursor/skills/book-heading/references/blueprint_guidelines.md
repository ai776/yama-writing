# Blueprint（章設計書）ガイドライン

## 1. Blueprint の目的

Blueprint は book-draft への「設計図」であり、以下の問題を防ぐ:

| 問題 | Blueprint での対策 |
|------|-------------------|
| 見出しの文言に本文が引きずられる | 「章の核心」と「禁止事項」で方向性を制御 |
| 批判的な見出しで他商品を攻撃しすぎる | 「禁止事項」で明示的に制限 |
| 章の目的が曖昧で散漫な本文になる | 「USP」「優先順位ルール」で焦点を固定 |
| 読者の前提知識を無視した展開になる | 「読者の現状認識」「壊す誤解」で起点を明示 |

## 2. 各フィールドの記入ルール

### 章の核心（1文）
- **何を:** この章で読者に伝えたいことを1文で書く
- **基準:** この1文だけ読めば章の存在意義がわかる
- **NG:** 「〜について説明する」（説明行為の記述は核心ではない）
- **OK:** 「不動産だけが銀行レバレッジを使える投資である」

### 読者の現状認識
- **何を:** この章を読む前の読者が「今こう思っている」状態を記述
- **基準:** ペルソナの知識・先入観・不安を具体的に書く
- **例:** 「株やNISAが最適な投資手段だと思っている」

### 本章で壊す誤解
- **何を:** 読者の現状認識のうち、この章で覆す部分を明示
- **基準:** 「〜という思い込み」の形式で書く
- **例:** 「株＝最適解という思い込み」

### 本章のUSP（独自の価値提案）
- **何を:** この章にしかない独自の切り口・価値
- **基準:** 他の章や他の書籍では語られていない角度
- **例:** 「銀行融資というレバレッジが使えるのは不動産だけ」

### 優先順位ルール
- **何を:** 本文で強調すべき要素の優先順序（1〜3位）
- **目的:** book-draft がどこに紙幅を割くべきかを制御する
- **例:**
  1. USP提示（最も重要）
  2. 比較は補助的に
  3. 批判は最小限

### 禁止事項
- **何を:** この章で絶対にやってはいけないこと
- **目的:** 見出しの文言に引きずられやすい方向を明示的に禁止
- **例:**
  - 他商品をギャンブル呼ばわりしない
  - 断定的な再現性保証をしない
  - 著者の体験以外の事例を捏造しない

### 必要要素
- **何を:** この章に必ず含めるべき具体的な要素
- **例:**
  - 具体例1件以上
  - 簡易比較図
  - 初心者保護の一言（リスク説明）
  - knowledge_base.md の K-XXX を使用

### 章のストーリー構造
- **何を:** 章内の情報提示順序を①〜⑤のステップで記述
- **基本パターン:**
  ① 誤解解除（読者の思い込みを壊す）
  ② 核心提示（この章のメインメッセージ）
  ③ 理由（なぜそう言えるのか）
  ④ 具体例（実際のケース）
  ⑤ 次章への橋渡し
- **カスタマイズ可:** 章の性質に応じてステップを変更してよい

## 3. Blueprint と見出しの整合性

### チェックポイント
- [ ] 見出し（H2）の順序がストーリー構造①〜⑤と対応しているか
- [ ] 核心トピック（`*`マーク）がUSPに対応しているか
- [ ] 禁止事項に反する見出しがないか（例: 「ダメな理由」→禁止: 批判的展開）
- [ ] 必要要素を収容する見出しが存在するか

## 4. 記入例

```markdown
# 第1章 Blueprint

■章の核心（1文）
不動産だけが銀行レバレッジを使える投資である。

■読者の現状認識
株やNISAは王道だと思っている。
不動産投資は怖い・自分には縁がないと感じている。

■本章で壊す誤解
株＝最適解という思い込み。

■本章のUSP
銀行融資というレバレッジ。他の投資商品では借入で投資できない。

■優先順位ルール
1. USP提示（融資レバレッジの独自性）
2. 比較は補助的に（株との違いは背景説明程度）
3. 批判は最小限（株を否定するのではなく、不動産の強みを提示）

■禁止事項
・他商品をギャンブル呼ばわりしない
・断定的な再現性保証をしない
・「株はダメ」という論調にしない（「不動産には〜がある」に変換）

■必要要素
・具体例1件（著者自身の初めての物件購入）
・簡易比較図（自己資金100万円 × 株 vs 不動産のリターン比較）
・初心者保護の一言（リスクも存在することの明示）
・K-001, K-003 のナレッジを使用

■章のストーリー構造
①誤解解除 — 「株やNISAが最適」という常識を問い直す
②核心提示 — 銀行融資レバレッジは不動産だけの特権
③理由 — なぜ銀行は不動産に融資するのか（担保価値）
④具体例 — 著者の100万円が1,000万円の投資になった話
⑤次章への橋渡し — 「では、どんな物件を選べばいいのか？」
```
