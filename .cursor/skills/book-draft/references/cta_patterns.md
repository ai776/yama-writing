# CTAの型パターン集

book-draft が本文執筆時に参照する、CTA（行動喚起）の設計パターン。
BookSpec の `goal.cta_flow` に基づき、最適な型を選択する。

---

## 基本原則

- CTAは **本文の文脈から自然に導かれる** こと（唐突な宣伝は厳禁）
- 読者が「もっと知りたい」「助けてほしい」と感じた瞬間にCTAがある設計
- 書籍内のCTAは **1〜3箇所** に絞る（多すぎるとセールス臭が出る）
- CTAの表現はAI禁止フレーズに抵触しないこと

---

## CTA配置の型

### 型1: エンディングCTA（最も基本）

**配置:** あとがきの最後、または最終章の末尾

**構造:**
```
本文の総括（読者の変化を振り返る）
  ↓
「ここから先は一人では難しい」という自然な境界の提示
  ↓
次のステップの案内（LINE登録/メルマガ/セミナー等）
  ↓
具体的なアクション指示（QRコード/URL）
```

**適するケース:** リード獲得、コンサル販売、セミナー誘導

**例文の方向性:**
- 「ここまで読んでいただいた方に、さらに一歩踏み込んだ○○をお伝えしています」
- 「実践の中で壁にぶつかったとき、いつでも相談できる場を用意しました」

---

### 型2: 章末ブリッジCTA（中盤で種を撒く）

**配置:** 核心トピックを扱った章（2〜3箇所）の章末

**構造:**
```
章の内容で読者の問題認知が深まった直後
  ↓
「この先をもっと詳しく知りたい方へ」という橋渡し
  ↓
軽いCTA（無料特典/補足資料のダウンロード等）
```

**適するケース:** 段階的にリード育成したい場合

**例文の方向性:**
- 「この章の内容をさらに掘り下げた○○を、LINE登録者限定で公開しています」
- 「ここで紹介した○○のチェックリストは、巻末のQRコードからダウンロードできます」

**注意:** 章末CTAは控えめに。本文の流れを切らないこと

---

### 型3: ストーリー埋め込みCTA（最も自然）

**配置:** 著者の体験談や事例紹介の中に溶け込ませる

**構造:**
```
著者or顧客の成功体験を語る
  ↓
「同じように変われた人たちがいる」という事実の提示
  ↓
「その場がここです」と自然に接続
```

**適するケース:** コミュニティ誘導、講座販売

**例文の方向性:**
- 「実は、この本をきっかけに○○を始めた方が、今では□□を達成しています。そんな仲間が集まる場を——」
- 「私のもとには毎週のように、『あの章を読んで行動しました』という報告が届きます」

**注意:** 情報源にない事例の創作は絶対禁止（operational_rules.md 遵守）

---

### 型4: 問題提起→解決予告CTA（恐怖→安心型）

**配置:** 問題を深く掘り下げた章の末尾

**構造:**
```
読者が「これはまずい」「自分も当てはまる」と感じる問題提示
  ↓
「でも、正しい方法を知れば防げる」という希望の提示
  ↓
「具体的な対処法をまとめた○○を用意しました」
```

**適するケース:** コンサル、診断サービス、個別相談への誘導

**例文の方向性:**
- 「もし今の状態を放置すると、○○のリスクがあります。でも、まだ間に合います」
- 「一人で判断するのが不安な方のために、無料の○○診断を用意しています」

## CTA導線パターン（BookSpec.goal.cta_flow との対応）

| 導線パターン | 書籍内CTA | 遷移先 | 最終ゴール |
|---|---|---|---|
| QRコード→LINE→ステップ配信→販売 | QRコード掲載 + 登録特典の案内 | LINE公式アカウント | バックエンド商品販売 |
| QRコード→LP→メルマガ登録 | QRコード掲載 + 無料資料の案内 | 専用LP | メルマガリスト構築 |
| URL→セミナー申込ページ | URL掲載 + セミナー内容の予告 | 申込フォーム | セミナー集客 |
| QRコード→診断ツール | QRコード掲載 + 診断の動機づけ | 診断ページ | 個別相談への接続 |
| 巻末→著者プロフィール→SNS | SNSアカウント掲載 | SNSプロフィール | フォロワー獲得 |

## CTA表現のNG/OK

| NG（セールス臭・AI臭） | OK（自然な誘導） |
|---|---|
| 「今すぐ登録してください！」 | 「興味を持たれた方は、こちらから詳しい情報をお届けしています」 |
| 「あなたの人生を変える第一歩です」 | 「ここまで読んで行動したいと思った方のために、次のステップを用意しました」 |
| 「限定○名！急いでください！」 | 「一緒に実践していく仲間が集まっています」 |
| 「さぁ、始めましょう！」 | 「まず試してみたい方は、○○からどうぞ」 |

## 執筆時の確認事項

- [ ] BookSpec の cta_flow に合ったCTA型を選んでいるか
- [ ] CTA箇所は本文の文脈から自然に導かれているか
- [ ] CTAの表現がAI禁止フレーズに該当していないか（banned_phrases.md 参照）
- [ ] CTA箇所が3箇所を超えていないか
- [ ] 情報源にない事例やデータをCTAの根拠に使っていないか
