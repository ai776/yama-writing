# タイトル案（book-title）

## スコアリング基準
- **SEO:** 検索ボリューム・キーワード適合性（/10）
- **訴求力:** 手に取りたくなる引力（/10）
- **ペルソナ適合:** 30-45歳サラリーマン初心者への響き（/10）
- **市場差別化:** 既存書籍との差別化（/10）

## 推奨

**案A「年収400万円から始める 中古アパート投資の教科書」** を推奨。

理由:
1. ペルソナ（30-45歳サラリーマン、年収400-800万）に最も直接的に刺さる
2. 「教科書」という位置づけがスクールへの導線と整合性が高い
3. SEOキーワードとの親和性が最も高い
4. 信頼感・安心感があり、初心者が手に取りやすい
