# StrategySpec -- 戦略設計書

## 事業ゴール

- **最終目的:** LINE登録 → 無料セミナー参加 → 不動産投資スクール入会（有料）
- **バックエンド商品:** 不動産投資スクール「脱しじり不動産」（累計2,400名、アクティブ2,000名在籍）。スクール内で融資情報データベース・物件情報・運営ノウハウを共有。月額/年額制の教育コミュニティ。
- **CTA導線:**
  1. 書籍を読む（問題認知 + 方向性の理解）
  2. 書籍末尾のQRコード → LINE登録
  3. LINE登録特典（PDF書籍・空室チェックリスト・スクール生購入物件一覧等）
  4. 無料セミナー（勉強会）への案内
  5. セミナー参加 → スクール説明 → 入会

## 想定読者（ペルソナ）

| 項目 | 内容 |
|------|------|
| 年齢層 | 28〜45歳 |
| 職業・立場 | サラリーマン（年収500万〜1,000万）。副業・資産形成に関心があるが、まだ不動産投資を始めていない、または始めたばかりで1棟目を探している段階 |
| 主な悩み | (1) 貯金はあるが投資先がわからない (2) 不動産投資に興味はあるが失敗が怖い (3) 何から始めればいいかわからない (4) 自分の年収で融資が通るか不安 (5) ワンルームマンション投資の勧誘を受けたが本当に大丈夫か疑問 |
| 価値観 | 堅実志向だが現状打破したい。「ギャンブル」ではなく「経営」として不動産投資に取り組みたい。再現性のある方法を求めている。ローリスク〜ミドルリスクで安定収入を得たい |
| 書籍を手にする場面 | Amazon Kindle/書店で「不動産投資 初心者」で検索。YouTubeチャンネルを視聴し著者に興味を持った。同僚や友人から「キムニーの本いいよ」と勧められた |

## KPI設計

| 指標 | 目標値 | 測定方法 |
|------|--------|---------|
| LINE登録数 | 読者の15%（初版3,000部 → 450件） | 書籍専用QRコード経由の登録数 |
| 無料セミナー参加率 | LINE登録者の30%（135名/月） | セミナー予約フォーム |
| スクール入会率 | セミナー参加者の10%（13〜14名/月） | 入会申込数 |
| Amazon評価 | 4.0以上（100レビュー時点） | Amazon評価ページ |

## 書籍の役割

- **主たる役割:** リード獲得装置（LINE登録 → セミナー → スクールへの導線の入口）
- **副次的役割:** 権威性証明（24棟保有・家賃年収5,000万円・スクール生2,000名の実績を書籍化することで信頼構築）

## 核心メッセージ（魂）

> サラリーマンこそ不動産投資の最強プレイヤーである。正しい知識と戦略（ロードマップ）さえあれば、年収500万・貯金500万からでも、中古アパートを軸に月100万円の不労所得を実現できる。
