## Part A: 戦略（StrategySpec）

### 基本情報

| 項目 | 内容 |
|------|------|
| 書籍タイトル（仮） | ※book-title で決定 |
| 著者名 | キムニー（脱しじり不動産） |
| 制作開始日 | 2026-02-18 |

### 事業ゴール

- **最終目的:** 不動産投資スクールへの集客（入会促進）
- **CTA導線:** 書籍末尾のQRコード → LINE登録 → 無料勉強会参加 → スクール入会
- **バックエンド商品:** 不動産投資スクール（2,000名規模、中古アパート・新築アパート投資の実践講座）

### 想定読者（ペルソナ）

| 項目 | 内容 |
|------|------|
| 年齢層 | 30〜45歳 |
| 職業・立場 | サラリーマン（年収400〜800万円）、不動産投資は未経験または初心者 |
| 主な悩み | 将来の収入不安、年金への不安、副収入が欲しいが何から始めればいいか分からない、不動産投資に興味はあるが失敗が怖い |
| 価値観 | 堅実だが現状を変えたい、リスクは避けたいが行動もしたい、本業以外の収入源に関心がある |
| 書籍を手にする場面 | 書店のビジネス書コーナー、Amazon/Kindleで「不動産投資 初心者」で検索、YouTubeチャンネル経由 |

### KPI設計

| 指標 | 目標値 | 測定方法 |
|------|--------|---------|
| LINE登録率 | 書籍購入者の15%以上 | QRコード経由のLINE友だち追加数 |
| 無料勉強会参加率 | LINE登録者の30%以上 | 勉強会申込数 |
| スクール入会CVR | 勉強会参加者の10%以上 | 入会申込数 |

### 書籍の役割

- **主たる役割:** リード獲得装置（LINE登録への誘導）
- **副次的役割:** 著者の権威性証明（24棟保有・家賃年収5,000万円の実績）

### 核心メッセージ（魂）

> 正しいやり方を知れば、年収が高くなくても自己資金が少なくても、中古アパート投資で月50〜100万円の不労所得を実現し、人生を逆転させることができる。

## Part B: 制作条件（ProductionSpec）

### 情報源タイプ

| 項目 | 内容 |
|------|------|
| source_type | youtube |

### トーン＆スタイル

- **基本文体:** 対話調（キムニーと聞き手の掛け合いを活かした語りかけスタイル）
- **口語許容度:** 高（「〜なんですよ」「〜じゃないですか」「めちゃくちゃ」等OK）
- **禁止表現（著者固有）:**
  - 「絶対に儲かる」「必ず成功する」等の断定的な利益保証表現
  - 金融商品取引法・景品表示法に抵触する表現
  - 他の投資家・不動産会社への誹謗中傷
  - YouTube動画のチャンネル登録促進文言（書籍には不要）

### 分量設計

| 項目 | 値 |
|------|-----|
| 総文字数目安 | 40,000〜45,000字 |
| 章数 | 8章（+ まえがき・あとがき） |
| 章ごと文字数予算 | 各章 4,500〜5,000字 |
| まえがき | 1,500字 |
| あとがき | 1,500字 |

### 図表・挿絵方針

| 項目 | 設定 |
|------|------|
| 挿入頻度 | 各章に1〜2点（合計10〜16点） |
| 図のタイプ | 比較表、フロー図、収支シミュレーション表、ロードマップ図 |
| 挿入位置ルール | 概念説明の直後または数値解説の補助として |
| キャプション | 「図X-Y: タイトル」形式 |

### 引用ルール

| source_type | 引用形式 |
|-------------|----------|
| youtube | 発言者名、動画タイトル、タイムスタンプ `[mm:ss]` |

**現在の設定:** 本文中に引用表記・出典表記は入れない（情報源は内部管理用として保持）
