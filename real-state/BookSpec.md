### 事業ゴール

- **最終目的:** LINE登録 → 無料セミナー参加 → 不動産投資スクール入会
- **CTA導線:**
  1. 書籍を読む（問題認知 + 方向性の理解）
  2. 書籍末尾のQRコード → LINE登録
  3. LINE登録特典（PDF書籍・空室チェックリスト・購入物件一覧等）
  4. 無料セミナー（勉強会）への案内
  5. セミナー参加 → スクール説明 → 入会
- **バックエンド商品:** 不動産投資スクール「脱しじり不動産」（累計2,400名、アクティブ2,000名在籍）

### 想定読者（ペルソナ）

| 項目 | 内容 |
|------|------|
| 年齢層 | 28〜45歳 |
| 職業・立場 | サラリーマン（年収500万〜1,000万）。副業・資産形成に関心があるが、まだ不動産投資未経験または1棟目を検討中 |
| 主な悩み | (1) 貯金はあるが投資先がわからない (2) 不動産投資に興味はあるが失敗が怖い (3) 何から始めればいいかわからない (4) 自分の年収で融資が通るか不安 (5) ワンルームマンション投資の勧誘を受けたが本当に大丈夫か |
| 価値観 | 堅実志向だが現状打破したい。「ギャンブル」ではなく「経営」としての不動産投資を求めている。再現性のある方法を重視 |
| 書籍を手にする場面 | Amazon/書店で検索、YouTubeチャンネル視聴から著者に関心、知人の推薦 |

## Part B: 制作条件（ProductionSpec より）

### 情報源タイプ

| 項目 | 内容 |

| source_type | youtube |

### トーン＆スタイル

- **基本文体:** です/ます調ベース、語りかけ風。著者が読者に直接話しかける親しみやすいトーン
- **口語許容度:** 高（「〜なんですよ」「〜じゃないですか」「ぶっちゃけ」「めちゃくちゃ」等OK。下品な表現・過度な若者言葉はNG）
- **禁止表現（著者固有）:**
  - 「絶対に儲かる」「必ず成功する」（断定的な利益保証）
  - 「誰でも簡単に」（安易さを煽る表現）
  - 「不労所得」の無条件使用（文脈なしでの使用は不可。著者は「経営」と位置づけ）
  - 特定金融機関の実名（イニシャル表記に）
  - スクール生個人の特定可能な情報

## プロデューサー判定（G1-P / G2-P 統合）

**判定: PASS**

StrategySpec と ProductionSpec の統合において矛盾は確認されず、以下の点で戦略的整合性が取れている:

1. **ペルソナ×トーン:** 28〜45歳サラリーマンに対して、YouTubeの親しみやすい語り口（口語許容度:高）は適切。堅すぎる「である調」では著者の人柄が出ない
2. **ゴール×分量:** リード獲得装置として50,000〜55,000字/10章は、読者が「方向性は理解したが、具体的な実行にはサポートが必要」と感じる適切な深さ
3. **核心メッセージ×章構成:** WHY→WHAT→HOW→ACTIONの4ブロック構成は、読者を「なぜ始めるべきか」から「具体的な一歩」へ自然に導く
4. **情報源×禁止表現:** YouTube動画の熱量を活かしつつ、利益保証表現は禁止。書籍としての信頼性を担保
