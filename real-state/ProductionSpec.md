| 項目 | 内容 |

| 書籍タイトル（仮） | サラリーマン大家の教科書 -- 年収500万・貯金500万から始める中古アパート投資 |

### YouTube版固有情報

- **対象チャンネル:** 脱しじり不動産（キムニー）
- **対象動画:** 18本の文字起こしテキスト（01〜18）
- **動画選定基準:** チャンネルの主要コンテンツを網羅的にカバー。初心者向け入門〜中級者向けロードマップまでを包含

## トーン＆スタイル

- **基本文体:** です/ます調ベース、語りかけ風。著者（キムニー）が読者に直接話しかけるような親しみやすいトーン
- **口語許容度:** 高（YouTubeの語り口を活かす。「〜なんですよ」「〜じゃないですか」「ぶっちゃけ」「めちゃくちゃ」等OK。ただし下品な表現や過度な若者言葉はNG）
- **禁止表現（著者固有）:**
  - 「絶対に儲かる」「必ず成功する」（断定的な利益保証表現）
  - 「誰でも簡単に」（安易さを煽る表現）
  - 「不労所得」の無条件使用（著者は「経営」であることを強調しており、文脈なしの「不労所得」は読者に誤解を与える）
  - 特定の金融機関名の実名記載（「R銀行」等のイニシャル表記にする）
  - スクール生個人の特定可能な情報

## 分量設計

| 項目 | 値 |
|------|-----|
| 総文字数目安 | 50,000〜55,000字 |
| 章数 | 10章 |
| 章ごと文字数予算 | 4,500〜5,000字 |
| まえがき | 2,000字 |
| あとがき | 2,000字 |

### 分量設計の根拠

- 18本の動画素材は非常に豊富（各動画30〜60分相当）で、テーマの幅が広い
- 10章構成で、各テーマを体系的にカバーしつつ、読者が段階的に理解を深められる構成とする
- 一般的なビジネス書（200〜250ページ）に適合する分量

## 図表・挿絵方針

| 項目 | 設定 |
|------|------|
| 挿入頻度 | 各章に2〜3点 |
| 図のタイプ | フロー図（投資ロードマップ、CTA導線）、比較表（新築vs中古、株vs不動産）、シミュレーション表（利回り計算、収支計算）、チェックリスト（物件選定、失敗パターン）、Mermaid図 |
| 挿入位置ルール | 概念説明の直後、または数値シミュレーションの補足として配置 |
| キャプション | 「図X-Y: タイトル」形式 |
